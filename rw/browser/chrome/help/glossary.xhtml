uru-buga rwizewe
Urubuga rukoresha imisobekere hamwe na &brandShortName; kugira ngo rubuze abandi bakoresha Interineti bangiriza kureba amakuru yoherejwe. Iyo usuye imbuga zizewe, &brandShortName; igaragaza agashushondanga k'ingufuri mu Mwanya Mimerere no mu Umwanyahantu. &brandShortName; na yo igaragaza izina ry'indangarubuga mu Mwanya Mimerere (kugira ngo ibuze imbuga zangiriza kwiba amakuru yawe) maze igahindura umuhondo imbuganyuma y'Umwanyahantu.
Imbuga zimwe zishobora kugirwa n'ibyatanzwe byizewe n'ibitizewe icyarimwe; ibyatanzwe bitizewe usanga atari iby'umuntu yihariye, nubwo nta cyizere ko ari uko biba bimeze. Kuri izo mbuga &brandShortName; izagaragaza agashushondanga k'ingufuri gaciyemo akarongo mu Mwanya Mimerere no mu Mwanyahantu kandi not izagaragaza izina ndangarubuga ry'urwo rubuga mu Mwanya Mimerere kugira ngo umenye ko paji rubuga igezweho itizewe burundu.

Dosiyenyibutsa
Ni agace gatoya k'amakuru kabitwe muri mudasobwa yawe n'imbuga makuru zimwe na zimwe. Iyo usuye rumwe muri izo mbuga, urubuga rusaba &brandShortName; gushyira Dosiyenyibutsa imwe cyangwa nyinshi mu bubiko disiki bwawe. Nyuma y'aho, iyo usubiye kuri urwo rubuga, &brandShortName; yoherereza urubuga Dosiyenyibutsa zarwo. Dosiyenyibutsa zigenda zibika amakuru akwerekeyeho, nk'ibiri mu kagorofani ujyana kugura ibintu. Ushobora kugena uburyo bw'Dosiyenyibutsa bwawe kugira ngo ucunge uko zikoreshwa n'ingano y'amakuru wifuza ko imbuga makuru zibikamo.
icy-emezo
Umubare ushobora gusibumbura indangamimerere. Icyemezo cyerekana izina ry'umuntu, ikigo, cyangwa undi muntu kandi kigahamya ko urufunguzo rusange, rwongewe muri icyo cyemezo ari urw'uwo muntu. Iyo ushyize umukonomubare ku butumwa cyangwa ibindi bintu, umukonomibare w'ubwo butumwa uremwa hifashishijwe urufunguzo rwihariye rujyanye n'urufunguzo rusange rwo mu cyemezo cyawe.
FIPS PUBS 140-1
Ibitangazwa n'Uburyo Bwemewe bw'Inonosorwa ry'Amakuru Mvantara (FIPS PUBS) 140-1 ni uburyo bwemewe bwa leta guverinoma ya Amerika bugenewe ikoreshwa ry'ibice by'imikonobanga — za apareye na porogaramumudasobwa busobeka kandi bugasobanura ibyatanzwe cyangwa bugakora ibindi bikorwa bijyana n'imikonobanga (nko kurema cyangwa kugenzura imikonomibare). Ibintu byinshi bigurwa na guverinoma ya Amerika bikomba kubahiriza uburyo bwemewe bwa FIPS bumwe cyangwa bwinshi.
uru-funguzo bwite
Imfunguzo zikoreshwa mu mukonobanga w'urufunguzo rusange. Urufunguzo bwite rugirwa ibanga kandi rukoreshwa kugira ngo hasobanurwe amakuru yasobetswe n'urufunguzo rusange bijyanye.

Ipaji rubuga ya XML irimo ilisiti y'amahuza yerekeza ku zindi paji rubuga. Porogaramu zihariye zishobora gusoma ilisiti y'ingingo z'amakuru zivuye mu mahuza, zigahita zivugurura ilisiti uko igenda ihinduka. Imbuga makuru zikoresha incamakerubuga kugira ngo zitangaze byihuse ingingo z'amakuru ashyushye, n'ibinyamakuru bwite byo kuri Interineti akenshi bikoresha incamakerubuga kugira ngo bimenyeshe ababisura ibyinjijwemo bishya. Iyo ukanze ihuza cyangwa ukandika URL kugira ngo ushake paji y'urubuga yihariye ubwihisho bufitiye kopi, &brandShortName; igereranya kopi yahishwe na kopi y'umwimerere. Iyo nta byahindutseho, &brandShortName; ikoresha kopi yahishwe aho kongera gushaka kopi y'umwimerere, bityo bikagabanya igihe cyo kunonosora no gukurura.
Interineti
Urusobemiyoboro rwo ku isi hose ruhuriweho na za miliyoni za mudasobwa zihana amakuru hakoreshejwe porotokole zemewe nka TCP/IP. Interineti mbere na mbere yahanzwe kugira ngo ikoreshwe mu mirimo ya gisirikare y'Abanyamerika mu w'1969, yaje gutera imbere inakoreshwa no mu mashuri no mu bigo by'ubushakashatsi noneho, mu mpera zo mu myaka y'1990, iza gukoreshwa na miliyoni z'ibigo by'ubucuruzi, imiryango n'abantu ku giti cyabo. Muri iki gihe Interineti ikoreshwa muri imeyili, mu gushakashaka amakuru ku Rubuga Makuru Mpuzamahanga, kohererezanya ubutumwa ako kanya, kohererezanya amakuru hagati y'abantu n'amatsinda y'abakoresha, no mu bundi buryo bwinshi.
i-jambobanga ngenga
Ijambobanga rikoreshwa kugira ngo barinde amagambobanga n'andi makuru bwite. &brandShortName; izahita ikwerekeza ku ijambobanga ngenga ryawe nushaka kugera kuri aya makuru. Niba ufite apareye z'umutekano zinyuranye, buri apareye izagusaba ijambobanga ngenga ritandukanye.
Java
Ururimi mpangaporogaramu rwahanzwe na Sun Microsystems. Porogaramu imwe ya Java ishobora gukoreshwa na mudasobwa nyinshi zitandukanye, bityo bigatuma abahanziporogaramu badakenera kurema verisiyo itandukanye ya buri porogaramu kuri buri mudasobwa. Nyuma yo kwinjizamo Java, &brandShortName; ishobora guhita ikurura no gutangiza porogaramu za Java (banita apuleti).
JavaScript
Ururimi nyandikaporogaramu rukunze gukoreshwa mu kubaka za paji rubuga. Abahanziporogaramu bakoresha JavaScript kugira ngo batume za paji rubuga zirushaho kuyega; JavaScript ikoreshwa kenshi mu kugenda bemeza amashusho no guhitamo buto. JavaScript ishobora gukoreshwa na Java, ariko ni ururimi rutandukanye ku rwego rwa tekiniki. Ntibisaba Java kugira ngo JavaScript ikore neza.
umu-kiriya
Porogaramumudasobwa (nka Internet browser) yoherereza ibibazo kandi ikakira ibisobanuro biturutse kuriseriveri, akenshi iba irimo gukorera kuri mudasobwa itandukanye. Mudasobwa porogaramumudasobwa y'umukiriya ikoreraho na yo ifatwa nk'umukiriya.
umu-konobanga

Aka-menyetso Myatsa
Ubwoko bwihariye bw'akamenyetso bukora nk'ububiko kugira ngo hajyemo amahuza mu ncamakerubuga. Ushobora kurema Akamenyetso Myatsa usura urubuga rufite incamakerubuga, ugakanda agashushondanga k'Akamenyetso Myatsa mu Mwanyahantu, maze ugahitamo incamakerubuga wifuza gukoresha.

Porotokole y'Iyiyimuradosiye (FTP)
Uburyo bwemewe butuma abakoresha bimura dosiye bazivana muri mudasobwa imwe zijya mu yindi binyujijwe mu rusobemiyoboro. Ushobora gukoresha &brandShortName; kugira ngo ushake dosiye ukoresheje FTP.
Porotokole Ngenzurahererekanya/Porotokole ya Interineti (TCP/IP)
Porotokole ya Yunigisi ikoreshwa mu guhuza mudasobwa zikorera kuri sisitemu mikorere zinyuranye. TCP/IP ni porotokole ya Interineti y'ingenzi bityo ikaba yarabaye uburyo bwemewe ku isi.
Uru-rimi Nsobeka Mwandikonyobora (HTML)
Imitererenyandiko ikoreshwa kuri pajirubuga. Uburyo bwemewe bwa HTML busobanura amatagi, cyangwa kode, zakoreshejwe kugira ngo hasobanurwe imigaragarire, imyandikire, imisusire, amashusho, n'ibindi bintu bigize pajirubuga.
Uru-rimi Nsobeka Ruhindurwa (XML)
Uburyo bwemewe budasobetse bwo gusobanura ibyatanzwe. Usibye HTML, XML utuma umuhanzi w'ipaji rubuga agena amatagi yihariye. Kugira ngo ubone ibindi bisobanuro, reba inyandiko W3C iboneka kuri Interineti Extensible Markup Language (XML).
seriveri
Porogaramumudasobwa (nka porogaramumudasobwa ikoreshwa kuri paji rubuga) yakira ibibazo bitanzwe na kandi ikohereza amakuru ku umukiriya, akenshi uba akorera kuri mudasobwa itandukanye. Mudasobwa porogaramumudasobwa ya seriveri ikoreraho nayo ifatwa nka seriveri.
mu-shakisha
Porogaramu ishingiye ku rubuga ituma abakoresha bashaka bakanabona amakuru yihariye avuye ku Rubuga Makuru Mpuzamahanga. Mushakisha ishobora gushaka inyandiko wuzuye w'inyandiko ziri ku rubuga cyangwa se ilisiti y'amagambo fatizo; ishobora na none gukoresha abakozi b'isomero basuzuma inyandiko zo ku rubuga bakazishyira ku rutonde n'intoki kugira zishobore kongera kuboneka. Ubusanzwe, ukoresha yandika ijambo cyangwa igicenteruro mu gasanduku shakisha, maze mushakisha ikagaragaza amahuza yerekeza kuri paji rubuga zishobora kugira akamaro.
i-sobanura
Igikorwa cyo gusobanura ibyatanzwe byasobetswe. Reba na none isobeka.
i-sobeka
Igikorwa cyo kuvangavanga amakuru mu buryo bujijisha icyo asobanura. Urugero, imyinjiro isobetse hagati ya za mudasobwa ituma bitorohera abandi bantu kuvangura, cyangwa gusobanura, amakuru anyura mu mwinjiro. Amakuru asobetse ashobora gusobanurwa gusa n'umuntu ufite urufunguzo rwabugenewe.

Ak-ugara ka Soketi Kizewe (SSL)
Porotokole ituma habaho iyemeza mpandezombi hagati y'umukiriya na seriveri kugira ngo bigirane umwinjiro wemejwe kandi usobetse. SSL ikorera hejuru ya TCP/IP no munsi ya HTTP, LDAP, IMAP, NNTP, n'izindi porotokole zifite urusobemiyoboro rwo mu rwego rwo hejuru. Uburyo bwemewe bushya bwa Internet Engineering Task Force (IETF) bwitwa Transport Layer Security (TLS) bushingiye kuri SSL. Reba na none iyemeza, isobeka.
